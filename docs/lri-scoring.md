# Lankside Readiness Index Scoring

The Lankside Readiness Index (LRI) is a deterministic, explainable readiness
indicator for Indian MSMEs. It is not a formal credit score, bank guarantee,
bank approval, legal certification, or government certification.

## Formula

```text
LRI = (0.4 x D) + (0.3 x C) + (0.3 x F)
```

- `D`: Document Integrity Score
- `C`: Collaboration Readiness Score
- `F`: Financial Consistency Score

The final score is rounded to two decimal places.

## Score Bands

- `RED`: 0 to below 50, not bank-ready
- `YELLOW`: 50 to below 80, needs improvement
- `GREEN`: 80 to 100, bankability ready

## Document Integrity Score

Document Integrity starts at 100 and is clamped between 0 and 100.

Deductions:

- 15 points for each missing critical document:
  - `GST_CERTIFICATE`
  - `UDYAM_CERTIFICATE`
  - `PAN_CARD`
- 10 points for each missing supporting document:
  - `BANK_STATEMENT`
  - `ITR_ACKNOWLEDGEMENT`
- 10 points for each failed mandatory document extraction
- 10 points for each unresolved `HIGH` severity audit finding
- 5 points for each unresolved `MEDIUM` severity audit finding
- 2 points for each unresolved `LOW` severity audit finding
- 5 points for each extraction confidence score below 0.60

## Collaboration Readiness Score

Collaboration Readiness starts at a 40 point baseline and is clamped between 0
and 100. It works even when the MOU module has not been implemented.

Additions:

- 20 points if at least one MOU record exists for the business
- 15 points if at least one MOU has status `GENERATED` or `EXPORTED`
- 10 points if any MOU has `cluster_purpose` filled
- 10 points if the business has `industry_type`
- 5 points if the business has both `city` and `state`

If no MOU table exists, only the baseline and profile completeness additions are
used.

## Financial Consistency Score

Financial Consistency starts at 100 and is clamped between 0 and 100.

Deductions:

- 20 points if `BANK_STATEMENT` is missing
- 20 points if `ITR_ACKNOWLEDGEMENT` is missing
- 15 points if bank statement extraction is missing or failed
- 15 points if ITR acknowledgement extraction is missing or failed
- 15 points for each unresolved PAN mismatch finding
- 10 points for each unresolved bank account holder mismatch finding
- 5 points if `turnover_range` is missing
- 5 points if `business_age_years` is missing

## Examples

Example 1:

- `D = 70`
- `C = 55`
- `F = 65`

```text
LRI = (0.4 x 70) + (0.3 x 55) + (0.3 x 65)
LRI = 28 + 16.5 + 19.5
LRI = 64.0
Band = YELLOW
```

Example 2:

- `D = 90`
- `C = 65`
- `F = 85`

```text
LRI = (0.4 x 90) + (0.3 x 65) + (0.3 x 85)
LRI = 36 + 19.5 + 25.5
LRI = 81.0
Band = GREEN
```

## Recommendations

Recommendations are generated by deterministic rules tied to deductions and
missing additions. Examples include:

- Upload missing GST, Udyam, PAN, bank statement, or ITR documents.
- Re-run extraction for failed or missing extraction results.
- Resolve PAN mismatch before applying for formal credit.
- Resolve bank account holder mismatch before lender submission.
- Fix high-severity audit findings first.
- Improve collaboration readiness by generating at least one structured MOU.

No AI model is used to calculate the final score or generate recommendations in
Phase 9.
